Adentro de la carpeta src se encuentra mi codigo, para compilarlo
se debe hacer lo siguiente:
1.-Abrir una terminal del tipo Linux.
2.-LLegar hasta la carpeta src con la terminal.
3.-Al estar dentro de la carpeta esribir en la terminal "make clean".
4.-Luego escribir "make".
5.-Luego escribir "./snakes".
6.-Con esto el codigo ya se estara ejecutando y ya podra jugar.
